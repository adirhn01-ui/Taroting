# Taroting 0.6.2

Two quality-of-life features. Free and open source, as always.

## New

- **Fullscreen playback** — press `F` (or the new button next to the transport) to watch your edit full-screen, movie-player style: play/pause, jump back/forward 5 seconds, a seek bar with the current position, and controls that fade away while playing and return on mouse movement. `Esc` exits. Arrow keys jump ±5 s while in fullscreen and go back to frame-stepping when you leave.

## Fixed

- **Home-screen covers for opened projects** — projects opened straight from File Explorer ("Open with Taroting" or double-click) showed a blank placeholder card on the home screen until a later save. The home screen now fills in missing covers automatically.

## Downloads

- **`Taroting-v0.6.2-portable-win64.zip`** — unzip and run `Taroting.exe`; no installation.
- **`Taroting_0.6.2_x64-setup.exe`** — per-user install with Start-menu entry, desktop shortcut, and `.trt`/media file associations. No admin rights needed. Windows SmartScreen may warn because the app is not yet code-signed — choose "More info" → "Run anyway".
